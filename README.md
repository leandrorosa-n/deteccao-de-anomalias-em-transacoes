# 🚨 Detecção de Anomalias em Transações Financeiras com Machine Learning

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de identificar possíveis fraudes em transações financeiras utilizando técnicas de **Machine Learning**, **Engenharia de Dados** e **Análise Preditiva**.

O desafio principal consiste em lidar com um cenário real de **dados altamente desbalanceados**, onde a quantidade de transações fraudulentas é muito menor quando comparada às transações legítimas.

Através da construção e avaliação de diferentes modelos, foi possível criar uma solução capaz de detectar comportamentos suspeitos e apoiar processos de prevenção a fraudes.

---

## 🎯 Objetivos

✅ Detectar transações potencialmente fraudulentas

✅ Trabalhar com bases de dados desbalanceadas

✅ Aplicar técnicas de Feature Engineering

✅ Comparar diferentes algoritmos de Machine Learning

✅ Melhorar métricas de Recall e Precisão

✅ Interpretar resultados e explicar decisões do modelo

## 📚 Principais Etapas do Projeto

### 1️⃣ Preparação dos Dados

* Importação da base de transações financeiras
* Análise da distribuição das classes
* Identificação do problema de desbalanceamento

### 2️⃣ Feature Engineering

* Criação de variáveis derivadas
* Transformação logarítmica de valores monetários
* Padronização dos dados utilizando StandardScaler

### 3️⃣ Treinamento dos Modelos

Foram avaliados diferentes algoritmos:

🔹 Logistic Regression

🔹 Random Forest

🔹 XGBoost

---

### 4️⃣ Balanceamento dos Dados

Implementação de técnicas para melhorar a detecção de fraudes:

📌 Undersampling

📌 Oversampling com SMOTE

📌 Ajuste de pesos das classes

---

### 5️⃣ Avaliação dos Resultados

Métricas utilizadas:

📈 Recall

📈 Precision

📈 F1-Score

📈 ROC-AUC

📈 Precision-Recall Curve

---

### 6️⃣ Otimização

* Ajuste de hiperparâmetros com GridSearchCV
* Definição de limiares personalizados para classificação
* Criação de pipelines para automação do fluxo de treinamento

---

### 7️⃣ Inteligência Explicável (XAI)

Utilização do SHAP para compreender:

🔍 Quais variáveis mais impactam as previsões

🔍 Como cada atributo influencia a decisão do modelo

🔍 Maior transparência e confiança nos resultados
